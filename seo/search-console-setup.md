# Google Search Console setup after launch

1. Add `urbanfresh.in` as a Domain property in Google Search Console.
2. Verify ownership using the DNS record Google provides.
3. Submit `https://urbanfresh.in/sitemap.xml`.
4. Inspect the home page and the four product landing pages, then request indexing.
5. Wait for real query data. New or rebuilt sites can take time to be crawled and measured.
6. Review Performance → Search results every month and record the useful findings in `monthly-log.csv`.

The website can be audited locally now, but ranking data only exists after the site is live, verified and receiving Google impressions.
